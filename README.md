# Titanic - Machine Learning from Disaster
Proyek ini merupakan implementasi machine learning untuk kompetisi Titanic: Machine Learning from Disaster di Kaggle. Tujuan utama dari kompetisi ini adalah memprediksi apakah seorang penumpang selamat atau tidak dari tragedi tenggelamnya kapal RMS Titanic berdasarkan fitur seperti nama, usia, jenis kelamin, kelas sosial, dan informasi lainnya.

## Tujuan Project
- Melakukan eksplorasi data untuk memahami pola keselamatan penumpang.
- Melakukan preprocessing dan feature engineering.
- Membangun model machine learning untuk prediksi keselamatan.
- Menghasilkan file submission.csv untuk dikirim ke leaderboard Kaggle.

## Dataset Description
Dataset terbagi menjadi dua bagian utama:
- train.csv	Digunakan untuk training model. Berisi fitur penumpang + label Survived.
- test.csv	Digunakan untuk melakukan prediksi pada data tanpa label.
